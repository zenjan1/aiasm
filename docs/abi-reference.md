# System V AMD64 ABI 参考

## 寄存器使用约定

### 参数传递（按顺序）
1. `rdi` - 第1个参数
2. `rsi` - 第2个参数
3. `rdx` - 第3个参数
4. `rcx` - 第4个参数
5. `r8` - 第5个参数
6. `r9` - 第6个参数

超过6个参数通过栈传递。

### 返回值
- `rax` - 整数返回值
- `rdx:rax` - 128位返回值

### 调用者保存（可能被破坏）
`rax`, `rcx`, `rdx`, `rsi`, `rdi`, `r8`, `r9`, `r10`, `r11`

### 被调用者保存（必须保留）
`rbx`, `rbp`, `r12`, `r13`, `r14`, `r15`

## 栈对齐
- 调用前栈必须16字节对齐
- call指令push 8字节返回地址
- 进入函数后rsp为8 mod 16
